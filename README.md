# AI-Final-Project

AI Art vs Human Art Classifier

This project uses a fine-tuned MobileNetV2 CNN model to classify images as AI-generated or human made.
All training, preprocessing, and evaluation code is included in the notebook.

Please download This data set: https://www.kaggle.com/datasets/kausthubkannan/ai-and-human-art-classification
You have to upload the dataset to YOUR google drive to run the code in Google Colab.
